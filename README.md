# 🚀 My Portfolio Project  

A modern, interactive, and fully responsive **3D AI-themed portfolio website** built with **Vite + TypeScript + React + Tailwind CSS + shadcn-ui**.  
This project showcases professional experience, freelance work, technical skills, and featured projects with immersive animations and a clean, futuristic design.

**🌐 Live Demo:** [Portfolio](https://darshak-portfolio.lovable.app/)

---

## 📌 Features  

✅ **3D Interactive UI** – Smooth animations, parallax effects, and 3D hover cards.  
✅ **Dark + Neon Theme** – AI-inspired gradients & cyberpunk accents.  
✅ **Dynamic Project Cards** – Clickable cards that open modals with descriptions, tech stacks, and GitHub links.  
✅ **Responsive Design** – Optimized for desktop, tablet, and mobile.  
✅ **Modern Tech Stack** – Built with the latest tools for speed and maintainability.

## 🚀 Getting Started

```sh
